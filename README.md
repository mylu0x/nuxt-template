## Getting started
Use [giget](https://github.com/unjs/giget) to create a new project:
```shell
yarn dlx giget@latest gh:mylu0x/nuxt-template <directory>
```

## Featured Directories & Files
- `uno.config.ts` - UnoCSS configuration file
- `nuxt.config.ts` - Nuxt configuration file
- `app/assets/css/main.css` - Main CSS file (includes reset CSS)
- `app/components/` - Components directory
- `app/layouts/default.vue` - Default layout file
- `app/pages/index.vue` - Home page (`/`)
